# 2. 벡터와 비트맵
| | 벡터 | 비트맵 |
|:-- | :-- | :-- |
| 장점 | 확대, 축소 시에 해상도에 영향을 주지 않는다. | 정교한 이미지가 가능하다. |
| 단점 | 정교한 이미지에는 부적합 | 확대, 축소 시에 해상도에 영향이 있다. <br> 고용량 문제 |
| 확장자 | AI,EPS,SVG | JPG, PNG, GIF, BMP |
| 사용 용도 | 로고, 아이콘, 일러스트 | 사진, 이미지, 그래픽 |
